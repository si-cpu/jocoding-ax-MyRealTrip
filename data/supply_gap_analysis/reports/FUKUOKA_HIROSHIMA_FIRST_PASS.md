# 후쿠오카·히로시마 관광 장소 공급 1차 분석

- 기준일: 2026-07-18
- 분석 단위: 투어 상품명이 아니라 실제 방문 장소
- 현재 MCP 표본: 후쿠오카 182개, 히로시마 26개
- 공식 장소 기준: 후쿠오카 5곳, 히로시마 29곳
- 수집 범위: 후쿠오카 투어 138개·입장권 31개·체험 13개, 히로시마 투어 13개·입장권 11개·체험 2개
- 투어 상세 범위: 원본 151행, 중복 제거 145개 상품 전부 조회 성공, MCP 상세 섹션 1,231개 확인
- 공개 전체 일정 범위: 145개 페이지 전부 조회 성공, 133개 상품에서 일정 149개·방문 슬롯 1,004개 확보
- 주의: 히로시마 액티비티 수집 일부가 429로 중단됐고 장소명 직접 검색 전이므로, 미연결은 상품 부재를 의미하지 않는다.

## 3. 카테고리 재수집 후에도 직접 검색이 필요한 장소

입장권 신·구 카테고리와 투어 상세 145개를 확인해 오리즈루 타워와 슛케이엔의 상품화가 추가로 확인됐다. 다음 시설은 재수집 결과에도 직접 연결되지 않았지만, 숨은 여행지로 확정하기 전에 한국어명·일본어명 개별 검색이 필요하다.

- 히로시마현립미술관
- 히로시마미술관
- 히로시마시 현대미술관
- 히로시마시 식물공원
- 히로시마시 아사 동물공원
- 5-Days 어린이문화과학관

이 장소들은 개별 장소명으로 직접 검색하고, 투어 후보가 나오면 상세 일정까지 확인한 후에만 공급 부족 여부를 판단한다.

## 4. 후쿠오카 공식 장소 기준

후쿠오카시는 BODIK에서 공개한 미술관·박물관 시설 5곳을 첫 공식 장소 기준으로 편입했다.

| 한국어명 | 일본어명 | 현재 판정 |
|---|---|---|
| 후쿠오카시미술관 | 福岡市美術館 | 카테고리 재수집 결과 직접 연결 없음·장소명 재검색 필요 |
| 후쿠오카아시아미술관 | 福岡アジア美術館 | 카테고리 재수집 결과 직접 연결 없음·장소명 재검색 필요 |
| 후쿠오카시박물관 | 福岡市博物館 | 카테고리 재수집 결과 직접 연결 없음·장소명 재검색 필요 |

현재 후쿠오카 공식 기준은 문화시설 5곳에 한정되므로, 공원·신사·전망시설·해변 등 도시 관광지 전체를 대표하지 않는다. 공식 장소 원천을 확장하기 전에는 도시 전체 공급률을 계산하지 않는다.

## 5. 다음 검증 순서

1. 후보 장소별 한국어명·일본어명 직접 검색
2. 히로시마 액티비티의 제한 중단 지점 재수집
3. 직접 검색에서 나온 투어는 상세 일정, 입장권은 상품 제목·대상 시설로 확정
4. 확인된 상품은 공급에 반영하고 남은 장소만 숨은 여행지 후보로 유지
