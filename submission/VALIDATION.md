## REST API 검증 범위

실제 Partner REST API 호출은 비밀 API 키를 대화나 로그에 남기지 않기 위해 수행하지 않았다.

대신 첨부된 공식 문서의 요청·응답 스키마를 기반으로 다음을 구현하고 모의 데이터로 검증했다.

- 투어: 카테고리, 검색, 상세, 날짜별 옵션
- 숙소: 지역 자동완성, 숙소 검색
- 항공: 공항 자동완성, 전체 목적지 최저가
- 인증: `MYREALTRIP_API_KEY` 환경변수만 사용

## 아직 검증하지 않은 항목

- 실제 API 키가 설정된 환경에서 Partner REST API 호출
- Codex 앱 설치 후 통합 스킬의 전체 자연어 실행
- 실제 API 키 환경에서 전체 목적지 국제선 최저가 조회
- 선택한 목적지의 MCP 실시간 항공편 후속 조회
- 날짜별 옵션이 있는 투어만 남기는 전체 자연어 실행
- 일반 추천 대비 새로운 후보 발견 증가 여부
- 실제 사용자의 링크 클릭과 예약 고려 의향

전체 목적지 항공 발견은 공식 Partner API가 제공하는 국제선 캘린더 최저가를 사용한다. 실제 좌석과 결제 가격은 목적지 선택 후 MCP 항공 검색으로 다시 확인해야 한다.

v0.4는 구조·판정 로직·공식 MCP 실데이터까지 검증된 프로토타입이며, Partner REST 실호출과 핵심 사용자 가치 가설까지 입증된 상태는 아니다.
