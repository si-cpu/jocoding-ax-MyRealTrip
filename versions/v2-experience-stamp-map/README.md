# v2 · T&A Experience Stamp Map

- 형태: Next.js + SQLite 웹 MVP
- 기준 문서 버전: Project Charter 0.2.0
- 상태: 제품 범위 확장 실험 보존본

## 주요 파일

- `PROJECT_CHARTER.md`: 여행 발견·타임라인·스탬프·스토리 설계
- `web/`: 실행 가능한 웹 MVP

## 실행

```bash
cd versions/v2-experience-stamp-map/web
npm install
npm run db:build
npm run dev
```

이 버전은 플러그인의 경험 지도를 여행 전·중·후 여정으로 확장했다. 이후 직무
초점이 넓어진다는 회고를 바탕으로 v3에서는 여행자 기능을 분리하고 사업개발
분석에 집중했다.
