## 3. 카테고리 재수집 후에도 직접 검색이 필요한 장소

입장권 신·구 카테고리와 투어 상세 145개를 확인해 오리즈루 타워와 슛케이엔의 상품화가 추가로 확인됐다. 다음 시설은 재수집 결과에도 직접 연결되지 않았지만, 숨은 여행지로 확정하기 전에 한국어명·일본어명 개별 검색이 필요하다.

- 히로시마현립미술관
- 히로시마미술관
- 히로시마시 현대미술관
- 히로시마시 식물공원
- 히로시마시 아사 동물공원
- 5-Days 어린이문화과학관

이 장소들은 개별 장소명으로 직접 검색하고, 투어 후보가 나오면 상세 일정까지 확인한 후에만 공급 부족 여부를 판단한다.

## 4. 후쿠오카 공식 장소 재분석

후쿠오카시는 공식 일본어 관광사이트 요카나비의 `観る・遊ぶ` 468건을 전체 원천으로 다시 수집했다. 이 원천은 삭제하지 않고 보존하며, 공식 한국어 가이드에도 동일 ID로 발행된 장소 중 후쿠오카시 범위와 방문 관광지 기준을 통과하고 통합 전 중복을 제거한 81곳을 한국인 여행자 대상 1차 분석 분모로 사용한다.

현재 MCP 표본과 입장권 제목·투어 상세 방문 근거를 교차한 결과 10곳이 연결됐다.

|---|---:|

잠정 확정 일치율은 `10/81 = 12.3%`다. 다만 MCP 일부 카테고리·페이지 오류가 남아 있으므로 이는 최종 일치율이 아니라 현재 확인 가능한 하한값이다. 단순 인접·조망·연결 설명과 조건부 대체 장소는 긍정 연결에서 제외했다. 전체 81곳의 일본어·한국어 병기와 연결 상태는 `data/supply_gap_analysis/reports/FUKUOKA_REANALYSIS.md`에 정리한다.
