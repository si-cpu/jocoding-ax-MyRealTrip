# 후쿠오카 MCP 역방향 장소 계층 감사

- 감사일: 2026-07-19
- 질문: MCP에는 있지만 공식 관광 데이터에는 없는 경험이 실제로 존재하는가?
- 결론: 단순 명칭 비교에서 보인 차이 대부분은 `하위 경험`, `다른 공식 카탈로그`, `시외 상품`이었다. 현재 확인된 명확한 민간 확장 후보는 `CLUB Rebellion`이며, 이것도 공식 관광지와 동일한 장소가 아니라 나카스 권역 내부의 사적 야간 체험이다.

## 왜 기존 비교가 틀어졌는가

기존 공식 분모는 요카나비의 `관광지(spots)` 목록이었다. 그러나 MCP 상품은 아래처럼 더 세밀하거나 다른 수준의 대상을 판매한다.

1. 권역·거리: 나카스, 나카강, 하카타강
2. 복합시설: BOSS E・ZO FUKUOKA
3. 하위시설: 팀랩 포레스트, 산리오 드리밍파크
4. 예약 경험: 리버 크루즈, 디너 크루즈, 호스트클럽 체험
5. 판매상품: 입장권, 옵션, 패키지

따라서 상품명과 관광지명을 1대1로만 비교하면 `복합시설 안의 하위 경험`과 `거리에서 운영되는 예약 체험`을 잘못 미연결로 판정한다.

## 수정된 매칭 단위

앞으로는 다음 계층을 각각 보존한다.

```text
도시
└─ 권역·거리·자연 구역
   └─ 테마파크·복합시설
      └─ 개별 장소·상점·시설
         └─ 예약 가능한 체험
            └─ 판매 상품·옵션
```

매칭 결과도 한 종류가 아니라 아래처럼 구분한다.

- `exact`: 같은 개별 장소 또는 체험
- `contained`: MCP 경험이 공식 상위 장소 안에 있음
- `official_other_catalog`: 관광지 목록이 아닌 공식 투어·체험·쇼핑 카탈로그에 있음
- `private_extension`: 공식 권역 안의 사적·상업 콘텐츠
- `outside_city`: 검색 도시는 후쿠오카지만 실제 장소는 시외
- `unresolved`: 운영사·주소 근거가 부족해 동일성 미확정

## 분석상 의미

`MCP만 있음`은 곧바로 마이리얼트립의 독자 콘텐츠로 간주하지 않는다. 먼저 공식 원본 전체, 공식 사이트의 다른 카탈로그, 상위 장소 포함 관계, 실제 주소를 차례대로 확인해야 한다. 반대로 이 과정을 통과한 `private_extension`은 지자체 관광지 목록이 잡지 못하는 민간 야간·체험 콘텐츠를 마이리얼트립이 확장한 사례로 별도 평가할 수 있다.

정확한 행 단위 근거는 `data/supply_gap_analysis/exports/fukuoka_mcp_reverse_hierarchy_audit.csv`에 보존한다.
