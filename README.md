# MyRealTrip 여행 경험 프로젝트

하나의 아이디어가 플러그인, 여행자용 웹 MVP, 사업개발 분석도구로 발전한 과정을
버전별로 보존한 저장소다. 각 버전은 독립적으로 읽고 실행할 수 있다.

## 버전 안내

| 버전 | 프로젝트 | 목적 | 상태 |
|---|---|---|---|
| [v1](versions/v1-codex-plugin/README.md) | 여행 경험 진열대 Codex 플러그인 | 상품보다 장소·음식·경험을 먼저 보여주고 공식 상품으로 연결 | 해커톤 제출본 |
| [v2](versions/v2-experience-stamp-map/README.md) | T&A Experience Stamp Map | 여행 발견부터 타임라인·스탬프·스토리까지 확장한 웹 MVP | 보존된 제품 실험 |
| [v3](versions/v3-ta-supply-gap-map/README.md) | T&A Supply Gap Map | MCP 상품과 공식 관광자산을 장소 단위로 대조하는 BD 분석도구 | 현재 분석 버전 |

## 발전 흐름

```text
v1 여행자 탐색 플러그인
→ v2 여행 전·중·후 웹 경험 확장
→ v3 T&A 사업개발용 공급 공백 분석으로 재정의
```

버전별 코드와 데이터는 섞지 않는다. 새로운 작업은 해당 버전 디렉터리 안에서
실행하며, 공통 루트에는 버전 인덱스만 둔다.
